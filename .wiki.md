# Project Summary
The LLM Chat Service is an interactive platform that allows users to engage with various AI models via the LiteLLM API. With a modern chat interface built using React and Vite, the service aims for high performance and scalability. It utilizes Kubernetes and Istio for effective deployment and traffic management, making it suitable for diverse applications in customer support, content generation, and more.

# Project Module Description
- **Chat Context**: Manages the state of the chat, including selected models and messages.
- **Model Selector**: User interface for selecting different AI models.
- **Chat Input**: Area for users to type their messages.
- **Chat Window**: Displays the conversation history and messages.
- **API Layer**: Handles communication with the LiteLLM API for model retrieval and message processing.
- **Deployment**: Involves Docker and Helm configurations, leveraging Istio for traffic management.

# Technology Stack
- **Frontend**: React, Vite, Tailwind CSS
- **API Communication**: Fetch API
- **Deployment**: Docker, Helm, Kubernetes, Istio

# Usage
1. **Environment Setup**: Create a `.envrc` and `.env.local` file with the necessary service and API configurations.
2. **Install Dependencies**: Run `pnpm install` in the project directory.
3. **Build the Application**: Use `pnpm run build` to compile the application.
4. **Run Docker Build**: Execute `./docker-build.sh` to build the Docker image.
5. **Deploy with Helm**: Use `./deploy.sh install` to deploy the application on Kubernetes.
6. **Development Server**: Start the development server using `pnpm run dev` to begin working on the application.
