# Project Summary
The LLM Chat Service is an interactive platform that facilitates user engagement with various AI models via the LiteLLM API. It is tailored for applications in customer support, content generation, and more, featuring a modern chat interface built with React and Vite. The service is deployed in a Kubernetes environment, ensuring high performance and scalability.

# Project Module Description
- **Chat Context**: Manages the chat state, including selected models and messages.
- **Model Selector**: Interface for selecting different AI models.
- **Chat Input**: Area for users to type messages.
- **Chat Window**: Displays conversation history and messages.
- **API Layer**: Handles communication with the LiteLLM API for model retrieval and message processing.

# Usage
1. **Environment Setup**: Create a `.envrc` file with the necessary service and API configurations.
2. **Install Dependencies**: Run `pnpm install` in the project directory.
3. **Build the Application**: Use `pnpm run build` to compile the application.
4. **Run the Application**: Execute `pnpm run dev` to start the development server.

### Docker Build Command
To build the Docker image, use:
```bash
docker build \
  --build-arg VITE_LITELLM_BASE_URL="https://openllm.your-domain.net" \
  --build-arg VITE_LITELLM_API_KEY="sk-your-key" \
  -t your-registry/llm-chat-service:latest \
  .
```

### Helm Deployment
To deploy with Helm, use:
```bash
helm install llm-chat-service ./deployments/helm \
  --namespace your-namespace \
  --create-namespace \
  --set image.repository=your-registry/llm-chat-service \
  --set image.tag=latest
```
