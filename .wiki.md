# Project Summary
The project is an LLM (Large Language Model) Chat Service that enables users to interact with various AI models through the LiteLLM API. It features a modern user interface for chatting with AI, allowing users to select models and send messages seamlessly. The service aims to provide a lightweight frontend application, utilizing Next.js and Vite for optimal performance and user experience, and is now enhanced with Kubernetes and Istio for deployment.

# Project Module Description
- **Chat Context**: Manages chat state, including selected models and messages.
- **Model Selector**: UI component for selecting the AI model.
- **Chat Input**: Input area for user messages.
- **Chat Window**: Displays chat messages and conversation history.
- **API Layer**: Handles communication with the LiteLLM API for fetching models and sending chat messages.
- **Deployment**: Includes Docker and Helm configurations for easy deployment, now integrated with Istio.

# Usage
1. **Environment Setup**: Create a `.envrc` and `.env.local` file with the following content:
   ```
   # Service Configuration
   SERVICE_HOST="llm-chat.internal.example.com"
   SERVICE_PORT="8080"
   
   # LiteLLM API
   VITE_LITELLM_BASE_URL=https://openllm.net
   VITE_LITELLM_API_KEY=your-api-key-here
   ```
2. **Install Dependencies**: Run `pnpm install` in the project directory.
3. **Build the Application**: Use `pnpm run build` to compile the application.
4. **Run Docker Build**: Execute `./docker-build.sh` to build the Docker image.
5. **Deploy with Helm**: Use `./deploy.sh install` to deploy the application on Kubernetes.
