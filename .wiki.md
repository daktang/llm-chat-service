# Project Summary
The LLM Chat Service is an interactive platform that allows users to engage with various AI models through the LiteLLM API. Designed for applications in customer support, content generation, and more, it utilizes a modern chat interface built with React and Vite. The service is deployed in a Kubernetes environment, ensuring high performance and scalability.

# Project Module Description
- **Chat Context**: Manages the chat state, including selected models and messages.
- **Model Selector**: Interface for selecting different AI models.
- **Chat Input**: Area for users to type messages.
- **Chat Window**: Displays conversation history and messages.
- **API Layer**: Manages communication with the LiteLLM API for model retrieval and message processing.
- **Deployment**: Involves Docker and Helm configurations.

# File Description Inventory
- **.envrc**: Contains environment variable configurations necessary for deployment and service settings.
- **Dockerfile**: Defines the multi-stage build process for the application, configured to serve via Nginx.
- **docker-build.sh**: Script to build the Docker image with specified environment variables.
- **deploy.sh**: Script to handle Helm deployment processes.
- **README.md**: Documentation for project setup and usage.
- **src/**: Contains all source code, including components, context, and pages.

# Technology Stack
- **Frontend**: React, Vite, shadcn/ui
- **API Communication**: Fetch API
- **Deployment**: Docker, Helm, Kubernetes

# Usage
1. **Environment Setup**: Create a `.envrc` file with the necessary service and API configurations.
2. **Install Dependencies**: Run `pnpm install` in the project directory.
3. **Build the Application**: Use `pnpm run build` to compile the application.
4. **Run Docker Build**: Execute `./docker-build.sh` to build the Docker image.
5. **Deploy with Helm**: Use `./deploy.sh install` to deploy the application on Kubernetes.
